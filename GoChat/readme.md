# 聊天室功能

## 資料夾分類
* cmd - 主要的應用程式執行
* controller - webapi服務
* config - 設定檔
* internal - 應用程序相關代碼(middleware、model、repository...等等)
* pkg - 共享庫, 包含一切其它項目的引用代碼
